# SmartStock Trading Platform - Frontend

- **Real-time Data**: Live stock prices and market updates
- **Portfolio Management**: Track holdings, P&L, and transaction history
- **Market Analysis**: Compare stocks, view charts, and market news
- **IPO Analysis**: Professional IPO risk assessments
- **Learning Center**: Educational content for trading and investing

## Tech Stack

- React 18 with TypeScript
- Vite for fast development and building
- Tailwind CSS for styling
- Radix UI components
- Chart.js for data visualization
- Socket.IO for real-time updates
- Axios for API calls

## Development

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
```

## Environment
